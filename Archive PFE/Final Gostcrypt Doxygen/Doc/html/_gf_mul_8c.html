◆ mul_x()

in_line void mul_x ( mode(32t)  x[4])

Multiply of a GF128 field element by x. The field element is held in an array of bytes in which field bits 8n..8n + 7 are held in byte[n], with lower indexed bits placed in the more numerically significant bit positions in bytes.

This function multiples a field element x, in the polynomial field representation. It uses 32-bit word operations to gain speed but compensates for machine endianess and hence works correctly on both styles of machine
