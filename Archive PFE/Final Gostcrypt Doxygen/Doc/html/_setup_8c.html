Default 'UninstallString' registry strings written by past versions of GostCrypt:

1.0 C: Setup.exe /u [optional] 1.0a C: Setup.exe /u [optional] 2.0 C: Setup.exe /u [optional] 2.1 C: Setup.exe /u [optional] 2.1a C: Setup.exe /u [optional] 3.0 C: Setup.exe /u [optional] 3.0a C: Setup.exe /u [optional] 3.1 The UninstallString was NEVER written (fortunately, 3.1a replaced 3.1 after 2 weeks) 3.1a C: Setup.exe /u 4.0 C: Setup.exe /u C: Files 4.1 C: Setup.exe /u C: Files 4.2 C: Setup.exe /u C: Files 4.2a C: Setup.exe /u C: Files 4.3 "C:\Program Files\GostCrypt\GostCrypt Setup.exe" /u C: Files\ 4.3a "C:\Program Files\GostCrypt\GostCrypt Setup.exe" /u C: Files\ 5.0+ "C:\Program Files\GostCrypt\GostCrypt Setup.exe" /u

Note: In versions 1.0-3.0a the user was able to choose whether to install the uninstaller. The default was to install it. If it wasn't installed, there was no UninstallString.
