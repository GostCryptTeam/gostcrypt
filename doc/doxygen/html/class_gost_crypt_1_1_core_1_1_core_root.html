Detailed Description

Class defining the core object containing methods for each action handled by GostCrypt (mount, dismount, create a volume, etc). This class is used by the process if it has root privileges. Therefore this class directly implement GostCrypt action. GostCrypt actions that do not require root privileges are inherited from the parent class CoreBase and the GostCrypt actions that require root privileges are implemented in this class.
