Detailed Description

Class storing all parameters for the volume header restore action This action will restore the header of the volume instance corresponding to given credentials. It will either use the internal backup or an external header backup file. For this action to succeed the password and keyfiles of the backup header should correspond to the password and keyfiles of the volume instance for which to restore the header TODO: Implement a way to restore the header when the password have changed.
